Help for the Lucid Browser Player

Version 1.1

Welcome to the Lucid Browser Player (JavaScript Edition), one member of the Lucid family of products. For more information on other Lucid and related programs, please visit the Lucidcentral website at www.lucidcentral.org

Written entirely in JavaScript, this latest edition of the Lucid Player is cross-platform, and will run within any modern web browser (Internet Explorer 8 and above, Edge, Chrome, Firefox, Safari and Opera.), under any operating system (such as Windows, Macintosh, Unix, Linux, Sun Solaris etc).

Unlike the older Java Applet or Desktop editions of the Lucid Player, the Java Runtime Environment (JRE) is not required.
