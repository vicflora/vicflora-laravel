Getting an Entities score value

Please note: This feature is not yet available in the JavaScript based Lucid Player

The Get Score function is useful for obtaining a quick view of how an entity in the current Lucid Key has been scored and how or if it relates to the entity you are attempting to identify.

To get the score information select the desired Feature/state and entity and click the Get Score button located on the main toolbar.

A get score dialog will then show the scoring information on the selected Entity for the given Feature or selected state.

Note: The Get Score function availability is set by the key author and may not be available in all keys.
